About this project

The Microsoft Cloud Stack

This platform is a comprehensive, interactive learning hub for the Microsoft cloud ecosystem — covering everything from foundational IT concepts to advanced Azure architecture, security, DevOps, and AI.

It's structured around 10 domains and 51 clusters, spanning the full depth of Microsoft cloud technologies. Every cluster contains theory, 3D architectural diagrams, step-by-step how-to guides with portal walkthroughs and PowerShell commands, real-world scenarios, and troubleshooting notes.

Why this exists

Microsoft Learn is comprehensive but flat. YouTube tutorials are scattered and outdated. Certification study guides cover exam objectives but not real-world implementation. There's no single place where you can understand a concept, see how it connects to the broader architecture, practice deploying it, and know exactly which career path it leads to.

This platform bridges that gap. Every topic connects to the bigger picture through interactive 3D diagrams. Every how-to guide is written from practitioner experience. Every cluster is mapped to real-world job roles so you always know why you're learning what you're learning.

How it's organised

The platform uses a three-tier structure: Domains (10 major areas like Compute, Networking, Identity & Security), Clusters (51 focused topic groups within those domains), and Topics (individual concepts and services within each cluster).

Content is also layered with two additional lenses. The service model layer maps clusters to IaaS, PaaS, SaaS, and Cloud Management categories. The career mapping connects clusters to 16 real-world job roles — from IT Support Engineer to Azure Solutions Architect.

The technology

The platform is built as a static site hosted on GitHub Pages with a custom domain. The 3D visualisations use Three.js with custom shaders, emissive materials, and OrbitControls. The diagram engine renders workflow, architecture, role-based, and feature diagrams from JSON data. The command palette provides fuzzy search across all 51 clusters and 200+ topics.

The visual design follows a deep-space glassmorphic aesthetic with Phantom Blue (#7DABFF) as the signature accent colour, backdrop-filter blur on all surfaces, and a persistent particle field that extends across every page.
