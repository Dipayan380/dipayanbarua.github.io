CAREER PATH

Azure Cloud Engineer

The broadest of the infrastructure-facing roles — you provision, secure, network, monitor, and automate across the whole platform. This path touches all nine core domains, in the order that builds correctly on itself.

17 clusters 9 domains ~85 hours AZ-104 · SC-900 aligned

Drag to rotate · click a node to jump straight into that domain

Who this is for

You want the generalist Azure skill set — the person a team calls first, who can provision compute, lock down identity, wire up networking, and keep the lights on, before specializing further. If you're completely new to IT, start with The Stack Floor first — this path assumes those fundamentals.

How this path works
Each cluster links to its real lesson page. Go in order the first time — later clusters assume the ones before them. Come back to specific clusters as reference once you've done a full pass.
