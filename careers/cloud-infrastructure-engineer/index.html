CAREER PATH

Cloud Infrastructure Engineer

You keep Azure environments running: provisioned, networked, monitored, backed up, and defined as code. This is the complete, ordered path — every cluster you need, in the sequence that builds on itself, nothing you don't.

13 clusters 6 domains ~65 hours AZ-104 · AZ-305 aligned

Drag to rotate · click a node to jump straight into that domain

Who this is for

You're comfortable at a terminal, curious about how systems fit together, and want to be the person a team trusts to keep infrastructure up, secure, and cost-sane. No prior Azure experience required — The Stack Floor domain covers the IT fundamentals underneath this path if you need them first.

How this path works
Each cluster below links to its real lesson page. Go in order the first time through — later clusters assume the ones before them. Once you've done a full pass, come back and use it as reference.

Optional, but worth it
Once you've completed the path, Terraform on Azure and Kubernetes (AKS) are the two clusters infra engineers reach for next most often — not required for this path, but a natural extension.

Certifications this path maps to

AZ-104: Azure Administrator AZ-305: Solutions Architect Expert

Where this leads

Cloud Infrastructure Engineer is a strong base for two directions: go deeper into automation and pipelines toward DevOps Engineer, or go broader into design and trade-offs toward Solutions Architect. Both build directly on everything above.
